# CHATTY

This is a full-stack web application built using the MERN stack, which includes **MongoDB**, **Express**, **React**, and **Node.js**. This project demonstrates my ability to work with modern web development technologies and build a scalable, interactive, and performant application.

## Features

- **User Authentication**: Implemented user authentication using JWT (JSON Web Tokens), allowing users to register, log in, and securely access their data.
- **CRUD Operations**: Provided full CRUD functionality for [resource, e.g., tasks, notes], where users can create, read, update, and delete items.
- **Responsive Frontend**: Built a responsive and dynamic frontend using React, enabling users to interact seamlessly with the application.
- **API Integration**: The backend exposes RESTful APIs that are consumed by the frontend using Axios.
- **State Management**: Used React hooks and context API for managing the state across components efficiently.
- **MongoDB**: Used MongoDB as a NoSQL database for storing user data and application-specific information.
  
## Technologies Used

- **Frontend:**
  - React
  - React Router
  - Axios for HTTP requests

- **Backend:**
  - Node.js
  - Express
  - MongoDB (via Mongoose)
  - JWT (JSON Web Tokens) for secure authentication
  - dotenv for environment variable management

- **Development Tools:**
  - Git & GitHub for version control
  - npm for package management
